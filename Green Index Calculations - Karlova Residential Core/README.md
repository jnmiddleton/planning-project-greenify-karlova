# Green Index Calculations — Karlova Residential Core

**Study area:** Residential Core, Karlova District, Tartu, Estonia  
**Language:** R (RStudio v2025.09.0-387, R v4.5.1)  
**Validation:** QGIS v3.28  

---

## Description

This folder contains the R Markdown script and knitted HTML output for calculating Green Index values across the historic residential core of Karlova, Tartu. The Green Index quantifies the percentage of greenery within each 100m × 100m grid cell, derived from binary classification of NDVI measurements, loosely following the methodology of:

> *Urban Neighborhood Green Index – A measure of green spaces in urban areas*  
> doi: [10.1016/j.landurbplan.2012.01.003](https://doi.org/10.1016/j.landurbplan.2012.01.003)

Green Index values are classified into four categories:

| % Green per Cell | Label |
|:----------------:|-------|
| < 25% | No Greenness |
| 25% – 49% | Low Greenness |
| 50% – 74% | Medium Greenness |
| ≥ 75% | High Greenness |

---

## Workflow

1. Load and reproject NDVI raster (TIFF, 10m × 10m) and study area polygon to EPSG:3857.
2. Clip NDVI to study area extent.
3. Create 100m × 100m aggregation grid.
4. Binary-classify NDVI (green = NDVI > 0.25) and extract mean greenness per cell.
5. Assign greenness labels from percentage thresholds.
6. Export final masked grid as GeoPackage.
7. Import GeoPackage into QGIS, manually delineate subregion boundaries with reference to land cover polygons and assign each grid cell to a subregion based on polygon containment.
8. Re-import subregion-attributed GeoPackage into RStudio for statistical testing.

---

## Statistical Testing

Greenness distributions were compared across subregions using non-parametric methods, as Shapiro-Wilk testing confirmed the log-transformed green percentage data was not normally distributed:

- **Kruskal-Wallis test:** Tests whether greenness levels differ significantly across subregions.
- **Dunn's post-hoc test (Bonferroni correction):** Identifies which specific subregion pairs differ significantly.
- **Compact letter display:** Annotated on boxplot; shared letters indicate no significant difference between subregion pairs.

## Notes

- Subregion boundaries were manually delineated in QGIS based on land use polygons and joined to Green Index cells by spatial containment. A residual category was created for cells that did not fall within any defined subregion boundary.
- The export step (`writeVector`) and Excel export (`write_xlsx`) are commented out in the script by default to prevent accidental overwrites.
- Results were validated visually in QGIS after each step in the workflow.
